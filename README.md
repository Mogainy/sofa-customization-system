# Sofa Customization and Ordering System

## Overview

The Sofa Customization and Ordering System is a web-based application that allows customers to customize sofas based on their preferred fabric, color, and design options before placing orders.

This project was developed using PHP, MySQL, HTML, CSS, and JavaScript.

## Features

* User Registration and Login
* Sofa Customization
* Fabric Selection
* Order Placement
* Order Management
* Customer Dashboard
* Responsive User Interface

## Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript
* XAMPP

## Database

The database file is included in this repository as:

`sofa_customizer.sql`

Import it into phpMyAdmin before running the project.

## Installation

1. Clone the repository.
2. Copy the project folder into XAMPP htdocs.
3. Import `sofa_customizer.sql` into MySQL.
4. Start Apache and MySQL.
5. Open:

http://localhost/sofa_customizer
